# Core temple ruins

Fixed Core Void POI pieces. `core_activation_dais.svg` marks the only valid World Engine activation tile. The other pieces form a deterministic ruined-temple footprint and should not be craftable player modules.

- `core_temple_ruins.svg`: combined establishing silhouette.
- `core_activation_dais.svg`: activation socket at `CORE`.
- `core_temple_arch_ruin.svg`: optional perimeter arch.
- `core_temple_pillar_broken.svg`: repeatable perimeter debris.
